### *LANGKAH PASANG MACRO DI EXCEL*

*1. Aktifin Tab Developer*

*2. Masukkin Kode Macro*  
Developer > Visual Basic > Insert > Module > Paste kode ini:
Sub PrintPDF_adto()
    Dim ws As Worksheet
    Dim NamaKlien As String
    Dim NoInvoice As String
    Dim TglInvoice As String
    Dim NamaFile As String
    Dim FolderPath As String
    
    Set ws = ThisWorkbook.Sheets("INVOICE") 'Ganti nama sheet kalo beda
    
    'Ambil data dari cell
    NamaKlien = Replace(ws.Range("B10").Value, "PT ", "") 'Cell nama klien
    NamaKlien = Replace(NamaKlien, " ", "")
    NoInvoice = Replace(ws.Range("F6").Value, "/", "") 'Cell no invoice
    TglInvoice = Format(ws.Range("B7").Value, "ddmmyyyy") 'Cell tanggal
    
    'Bikin nama file otomatis
    NamaFile = "INV_ADTO_" & NamaKlien & "_" & TglInvoice & ".pdf"
    
    'Folder simpan - ganti sesuai folder kamu
    FolderPath = CreateObject("WScript.Shell").SpecialFolders("Desktop") & "\Invoice ADTO\"
    
    'Bikin folder kalo belum ada
    If Dir(FolderPath, vbDirectory) = "" Then MkDir FolderPath
    
    'Area print: A1:G45, sesuaikan sama templatemmu
    ws.PageSetup.PrintArea = "A1:G45"
    ws.PageSetup.Orientation = xlPortrait
    ws.PageSetup.Zoom = False
    ws.PageSetup.FitToPagesWide = 1
    ws.PageSetup.FitToPagesTall = 1
    
    'Export ke PDF
    ws.ExportAsFixedFormat Type:=xlTypePDF, Filename:=FolderPath & NamaFile, _
        Quality:=xlQualityStandard, IncludeDocProperties:=True, _
        IgnorePrintAreas:=False, OpenAfterPublish:=True
        
    MsgBox "PDF berhasil dibuat: " & vbCrLf & NamaFile, vbInformation, "PT Adto Cipta Usaha Mandiri"
End Sub
*3. Bikin Tombol di Sheet Invoice*

### *TIPS FOLLOW UP BIAR NGGAK DIANGGAP SPAM:*

1. *Timing*: H+3, H+7, H+14. Lewat itu udah cold, mending telepon.
2. *Selalu Kasih Value Baru*: Update stock, harga khusus, free konsul teknis. Jangan cuma "nanya kabar".
3. *WA Follow Up*: 1 jam setelah kirim email, WA: "Pagi Pak, barusan saya email follow up quotation Bentonite ya. Mohon dicek 🙏"
4. *Telpon*: Kalau H+7 belum reply, telpon langsung. Bilang: "Mau mastiin email saya masuk aja Pak, takutnya ke spam"
5. *Tracking*: Pake Mailtrack/Gmail read receipt biar tau udah dibaca apa belum.
